# 🎯 Jogo de Adivinhação

## 📌 Descrição

Um jogo de terminal onde o usuário tenta adivinhar um número gerado aleatoriamente pelo sistema, com base em níveis de dificuldade.

## 🎮 Como funciona

1. Escolha o nível de dificuldade
2. O sistema define um intervalo de números
3. Você tem um número limitado de tentativas
4. Recebe dicas: "muito alto" ou "muito baixo"
5. Ganha ao acertar o número, perde ao acabar as tentativas

## 🧠 Níveis de dificuldade

| Nível   | Intervalo | Tentativas |
| ------- | --------- | ---------- |
| Fácil   | 0–49      | 10         |
| Médio   | 0–99      | 7          |
| Difícil | 0–499     | 5          |

## ▶️ Como executar

```bash
go run main.go
```

## 📚 Conceitos utilizados

- Condições (if/else)
- Loops (laços)
- Estrutura switch
- Impressão de saídas em diferentes formatos
- Declaração de variáveis
